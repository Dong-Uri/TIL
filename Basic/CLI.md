# CLI

CLI ↔ GUI

- GUI가 더 편하고 쉽지만 CLI가 더 적은 리소스를 사용, 더 다양한 기능

- GUI
    - 그래픽을 통해 사용자와 컴퓨터가 상호 작용하는 방식

- CLI
    - 명령어를 통해 사용자와 컴퓨터가 상호 작용하는 방식

## 명령어

- touch
    - 파일을 생성하는 명령어
- mkdir
    - 새 폴더를 생성하는 명령어
- ls
    - 현재 작업 중인 디렉토리의 폴더/파일 목록을 보여주는 명령어 (dir과 유사)
- cd
    - 현재 작업 중인 디렉토리를 변경하는 명령어
    - cd .. : 상위폴더로 이동
- start (mac : open)
    - 폴더/파일을 여는 명령어
    - . : 현재위치를 의미
- rm
    - 파일을 삭제하는 명령어
    - -r 옵션을 주어야 폴더 삭제 가능
- and more…
