# Jupyter Notebook

- 종료
    - bash에서 ctrl + c
    - 그냥 꺼도 됨
- H
    - 단축키 목록 출력
- A, B
    - 상단, 하단 셀 생성
- X, DD
    - 셀 삭제
- ctrl, shift, alt + enter
    - 셀 실행, 셀 실행 후 다음 셀 선택, 셀 실행 후 다음 셀 생성
- M, Y
    - 마크다운 셀로 변경, 코드 셀로 변경

# Pycharm

- shift + f10
    - 코드 실행
- shift + f9
    - 디버그 실행
- breakpoint
    - 확인할 곳 다음 줄에 검
    - 우 클릭으로 조건을 넣울 수도 있음
- f7
    - 한 줄씩 디버그
    - 함수가 나오면 함수 안으로
- f8
    - 한 줄씩 디버그
- shift + f8
    - 한 줄 위로
- f9
    - 다음 breakpoint까지 resume
