# Jupyter Notebook

- 종료
    - bash에서 ctrl + c
    - 그냥 꺼도 됨
- H
    - 단축키 목록 출력
- A, B
    - 상단, 하단 셀 생성
- X, DD
    - 셀 삭제
- ctrl, shift, alt + enter
    - 셀 실행, 셀 실행 후 다음 셀 선택, 셀 실행 후 다음 셀 생성
- M, Y
    - 마크다운 셀로 변경, 코드 셀로 변경
