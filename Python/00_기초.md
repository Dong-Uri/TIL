# 코드 작성
- [PEP8](https://www.python.org/dev/peps/pep-0008/)
  - 파이썬에서 제안하는 스타일 가이드
- [Google Style guide](https://google.github.io/styleguide/pyguide.html)
  - 각 회사/프로젝트 마다 따로 스타일 가이드를 설정하기도 함
- 들여쓰기
  - space키 4번 (PEP8 권장)
  - tab키 1번 (교수님 권장)
  - **혼용금지**
- 주석
  - `# 주석`
  - 여러줄 주석
    - 한줄 씩 `#` 사용
    - `''' ''', """ """`로 묶음

# 변수
- 알파벳, _, 숫자로 구성
- 첫 글자는 숫자 X
- 대소문자 구별
- 예약어 사용 불가
  - `keyword.kwlist`로 확인
