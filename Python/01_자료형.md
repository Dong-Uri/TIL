# 수치형 (Numeric Type)

## 정수 자료형 (int)

- 진수 표현 가능
  - 2진수
    - `0b`
  - 8진수
    - `0o`
  - 16진수
    - `0x`

## 실수 자료형 (float)

- 부동 소수점 조심할 것
  - 매우 작은 수로 확인
  - math 모듈 활용
    - `math.isclose(a, b)`

# 문자열 자료형 (String Type)

- 여러줄 입력시 삼중 따옴표 사용
- String Interpolation
  - 왠만하면 f-string 사용

# None

- 없음

# 불린형 (Boolean)

- 논리 연산자의 단축 평가
  - `5 and 4 -> 4`
  - `5 or 3 -> 5`
